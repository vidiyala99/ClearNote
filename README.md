# ClearNote

ClearNote is a clinical documentation app with a React frontend and a FastAPI backend. It lets clinicians record visits, upload audio for transcription, and review structured summaries generated by background workers.

## Repository Layout

- `frontend/`: Vite + React + TypeScript client.
- `backend/`: FastAPI API, SQLAlchemy models, Alembic migrations, and Celery workers.
- `docs/`: design docs and implementation plans.
- `docker-compose.yml`: local development stack for API dependencies and workers.

## Current Product Surface

- Public landing page with auth entrypoints.
- Guest-accessible demo dashboard for previewing the app without signing in.
- Authenticated visit recording flow.
- Visit detail screen for transcript and AI summary review.
- Background processing with Celery, Redis, and Postgres-backed persistence.

## Tech Stack

- Frontend: React 18, TypeScript, Vite, React Router, Clerk, Tailwind CSS.
- Backend: FastAPI, SQLAlchemy, Alembic, Celery, Redis, Postgres.
- Integrations: Clerk auth, S3 uploads, OpenAI-based summarization.

## Prerequisites

- Node.js 18+
- npm
- Python 3.12+
- Docker Desktop or a local Postgres + Redis setup

## Environment Setup

1. Copy `.env.example` to `.env`.
2. Fill in the required values:
   - `DATABASE_URL`
   - `REDIS_URL`
   - `CLERK_JWKS_URL`
   - `AWS_ACCESS_KEY_ID`
   - `AWS_SECRET_ACCESS_KEY`
   - `S3_BUCKET_NAME`
   - `AWS_REGION`
   - `CORS_ORIGINS`
   - `OPENAI_API_KEY`

`backend/app/config.py` reads the root `.env` file directly.

## Running Locally

### 1. Start infrastructure

```bash
docker compose up -d postgres redis
```

If you want the full backend stack in Docker, run:

```bash
docker compose up --build
```

### 2. Run the backend locally

From `backend/`:

```bash
pip install -e .[dev]
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 3. Run the worker locally

From `backend/`:

```bash
celery -A app.workers.celery_app worker --loglevel=info
```

Optional beat process:

```bash
celery -A app.workers.celery_app beat --loglevel=info
```

### 4. Run the frontend

From `frontend/`:

```bash
npm install
npm run dev
```

The frontend runs on `http://localhost:5173` and proxies API requests to `http://localhost:8000`.
